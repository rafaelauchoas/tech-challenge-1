# PÓS TECH FIAP - Data Analytics | Tech Challenge 1
Este projeto faz parte do primeiro Tech Challenge da Trilha de Data Analytics da PÓS TECH da FIAP e tem como objetivo preparar e estruturar o **Brazilian E-Commerce Public Dataset by Olist** para análise no **Power BI**, com foco na construção de um **relatório executivo voltado a investidores e acionistas do setor de e-commerce**.

A proposta é transformar dados transacionais brutos em uma base limpa, consistente e modelável, capaz de sustentar análises posteriores.

## Objetivo do projeto

O projeto utiliza o dataset público da Olist como fonte de dados e aplica um pipeline de ETL em Python para:

- baixar os arquivos
- limpar e padronizar as tabelas
- enriquecer colunas relevantes para análise
- exportar tabelas tratadas em formato ideal para modelagem no Power BI
